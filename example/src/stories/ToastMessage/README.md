# ToastMessage
The `ToastMessage` renders a responsive box-model layout component.

<!-- STORY -->

### Usage
```jsx
import { ToastMessage } from 'rimble-ui'
```

<!-- component example here -->
```jsx
<ToastMessage
  p={5}
  fontSize={4}
  width={[ 1, 1, 1/2 ]}
  color='white'
  bg='salmon'>
  This is a ToastMessage
</ToastMessage>
```
