# Avatar
A basic avatar component.

<!-- STORY -->

### Usage
```jsx
import { Avatar } from 'rimble-ui'
```

<!-- Avatar example here -->
```jsx
<Avatar />
```

<!-- Large,Medium, and Small Avatars here -->
```jsx
<Avatar size='small' />
<Avatar size='medium' />
<Avatar size='large' />
```

### Props
| Name   | Type   | Default | Description                                 |
| ------ | ------ | ------- | ------------------------------------------- |
| `size` | string |         | Sets avatar size to small, medium, or large |
